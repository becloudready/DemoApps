# DemoApps

A collection of small demo and learning projects.

## Apps

| App | Description | Stack |
|-----|-------------|-------|
| [world-clock](./apps/world-clock) | World clock with multiple timezones | React |
| [employee-app](./apps/employee-app) | Full-stack employee management CRUD app | React + FastAPI + MongoDB |

## Getting started

Each app is self-contained. Navigate into an app folder and run:

```bash
cd apps/<app-name>
npm install
npm start
```

## Structure

```
apps/
├── world-clock/     # React world clock app
└── employee-app/    # React + FastAPI + MongoDB employee CRUD app
```
